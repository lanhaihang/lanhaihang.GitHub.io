串联所有单词的子串

@Author: YiHua Lee @Address: Guangdong province, China

题目

LeetCode中第30题：https://leetcode-cn.com/problems/substring-with-concatenation-of-all-words/

题目难度：困难

  1. 给定一个字符串 s 和一些长度相同的单词 words。
  2. 找出 s 中恰好可以由 words 中所有单词串联形成的子串的起始位置。
  3. 子串要与 words 中的单词完全匹配，中间不能有其他字符。
  4. 不需要考虑 words 中单词串联的顺序。

解答

解题思路

  1. 用两个 HashMap 来解决这个问题。
  2. 首先，我们把所有的单词存到 HashMap 里，key 直接存单词，value 存单词出现的个数。
  3. 然后扫描子串的单词，如果当前扫描的单词在之前的 HashMap 中，就把该单词存到新的 HashMap 中。
  4. 判断新的 HashMap 中该单词的 value 是不是大于之前的 HashMap 该单词的 value。
  5. value如果大了，就代表该子串不是我们要找的，接着判断下一个子串就可以了。
  6. 子串扫描结束，如果子串的全部单词都符合，那么该子串就是我们找的其中一个。

实现

时间复杂度：假设 s 的长度是 n，words 里有 m 个单词，那么时间复杂度就是 O（n * m）。

空间复杂度：两个 HashMap，假设 words 里有 m 个单词，就是 O（m）。

大佬题解

解题思路

  1. 假设words数组长度为L，word单词长度为WL，遍历字符串s，下标记做i

  2. 需要比对的单词起始坐标则为 [i, i+WL, i+2WL ... i+(L-1)*WL]

  3. 如果i满足条件，各个单词的第k位之和一定相等

    即：words[0][k] + words[1][k] + ... + words[L-1][k] == s[i + k] + s[i+WL + k] + ... + s[i+(L-1)*WL + k]

  4. 反之，若对于i，满足后者条件的i则可能为正确结果，这个时候直接校验即可。
